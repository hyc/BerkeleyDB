Berkeley DB: DbEnv.get_lg_info

DbEnv.get_lg_info

import com.sleepycat.db.*;


public DbLog get_lg_info();

Description

Get/set the return value from the DbLog.open method, used if modifications to the file being opened should be logged. If no value is set, no logging is done by the Berkeley DB access methods.

Class

DbEnv
