Berkeley DB: DbInfo.set_re_pad

DbInfo.set_re_pad

import com.sleepycat.db.*;


public void set_re_pad(int re_pad);

Description

Set the pad character for short records for the Recno access method. The pad character is used for fixed length records, if the DB_PAD flag is set. If the DB_PAD flag is not explicitly set, <space> characters (i.e., 0x20) are used for padding.

Class

DbInfo
